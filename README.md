# Приложение приоритизации источников

Консольное CLI-приложение для пользовательской перекрёстной приоритезации источников информации, описанных в markdown-файлах с frontmatter.

## Установка и запуск

Приложение написано на Python 3.10+ и не требует установки дополнительных зависимостей.

```bash
# Запуск приложения
python src/main.py --path ./sources

# Показать количество непрочитанных источников по типам
python src/main.py --path ./sources --count
```

## Аргументы командной строки

- `--path` (обязательный): Путь к директории с markdown-файлами источников
- `--count` (необязательный): Показать количество непрочитанных источников по типам и завершить работу

## Формат файлов источников

Файлы источников должны быть в формате Markdown и содержать:

1. Frontmatter в формате YAML в начале файла, ограниченный `---`:
   - `type`: тип источника (статья, книга, видео и т.д.)
   - `readed`: булево значение (`true`/`false`)

2. Первая строка после frontmatter должна содержать название источника в формате:
   - `- [ ] Название источника` или
   - `- [/] Название источника`

Пример файла источника:

```markdown
---
type: статья
readed: false
---
- [ ] Интересная статья о машинном обучении
Содержимое статьи...
```

## Алгоритм приоритезации

Приложение реализует двухэтапный алгоритм приоритезации:

### Этап 1: Групповой отбор (для >50 источников)
- Источники разбиваются на группы размером alpha = ceil(count / 25)
- У пользователя запрашивается выбор самого важного источника в каждой группе
- Не выбранные источники сразу получают статус "lowest"

### Этап 2: Парные сравнения
- Оставшиеся источники сравниваются попарно
- За каждую победу начисляется 1 балл
- По итогам.sources разделяются на 5 равных групп и получают статусы:
  - highest (🔺) - лучшие 20%
  - high (⏫) - следующие 20%
  - medium (🔼) - следующие 20%
  - low (🔽) - следующие 20%
  - lowest (⏬) - худшие 20%

## Результат работы

После завершения приоритезации, файлы источников обновляются:
- Для источников со статусом "highest" строка названия изменяется на: `- [/] Название источника 🔺`
- Для остальных статусов: `- [ ] Название источника [маркер]`

Маркеры статусов:
- 🔺 - highest
- ⏫ - high
- 🔼 - medium
- 🔽 - low
- ⏬ - lowest
